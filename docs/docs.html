Ethan

Ethan, a friend of mine. He's from America. He's good at crafting. He can make many different things with wood. I like him.
We've never met. I don't know if we can ever meet. But still we've a god friendship. He always says that I'm nota techy guy. And some people think that programming is too hard and programmers are aline. So, I thought to make something where people can code in their mother tounge. And it's Ethan.

My mother tounge is Bangla. I'm not so fluent in English. I suffered a lot when I was new to the programming world. Everything is in English: the documantation, the articles, the tutorials even the code itself. I found this was a big cause why people think that coding is hard.
To make a change, I created Ethan. It's programming language where people can write code in their own language. Currently it only supports Bangla and English sysntax but I hope one day it'll available in all languages.

This is basically a source to source compiler (transpiler). It's designed to produce C code from our source. My aim will be produced safe and opmtimized C code but I don't know how much I can do.
Currently, it's missing many basic features of a programming language like functions and string (data-type). I hope people will contribute and make it complete.

Ready to take of??

We can write code anywhere, I mean anywhere (even in a stone). But only writing codes will do nothing unless you can give instruction to the computer with it. To make computers to do something with your own code, we need to talk with it in its own language which is 1 and 0. But it's hard to do. So, people have been inventing programming languages which is like human languages. And to convert the human languages into machine language, we can use an interpreter or a compiler or a transpiler.

To get started with Ethan, you need to download (click me) the transpiler first. After downloading the .zip file, you need to extract it. After extracting it you need to run the Ethan.py file. Then you are ready to go.
